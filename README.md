NeuroFlow
==============================


## Installation

### Virtual Environment

#### Conda
`conda env create -f environment.yml` <br>
`conda activate neuroflow` <br><br>

#### Venv
`python -m venv neuroflow` <br>
`source neuroflow/bin/activate` (Linux/MacOS) <br>
`neuroflow\Scripts\activate` (Windows) <br>
`pip install poetry`


### Dependencies
#### Poetry
`poetry install`


### Package
`pip install -e .`
